Linear Equation In Two Variables

1. Introduction to Linear Equation In Two Variables

What is linear equation in two variables?

An equation in the form ax + by + c = 0, where a, b and c are real number such that a is not equal to 0 and b is not equal to 0, is called a linear equation in two variables x and y.

For example, each of the following are linear equations in two variables x and y,

  1. 2x + 3y - 12 = 0
  2. 4x - y + 5 = 0
  3. √2x - √3y - 9 = 0

Solutions of Linear Equations

Let ax + by + c = 0 be a given linear equation in x and y. Then a pair of values, one for x and one for y, which satisfy the given equation is called is solution.
If x = α and y = β satisfy the equation ax + by + c = 0 then we say that the ordered pair (α,β) is its solution.

Examples Regarding The Topic

Example 1: Write each of the following equations in the form ax + by + c = 0 and indicate the values of a, b and c in each case.

  1. x = 4y
  2. x/3 - y/2 = 5
  3. 4y - 3
  4. πx + y = 6

Solution:

(i) x = 4y ⇒ x - 4y = 0
⇒ x - 4y + 0 = 0.
This is of the form ax + by + c = 0, where a = 1, b = -4 and c = 0.

(ii) x/3 - y/2 = 5 ⇒ 2x - 3y = 30.
This is of the form ax + by + c = 0, where a = 2, b = -3 and c = -30.

(iii) 4y - 3 = √2x ⇒ √2x - 4y + 3 = 0.
This is of the form ax + by + c = 0, where a = √2, b = -4 and c = 3.

(iv) πx + y = 6 ⇒ πx + y - 6 = 0.
This is of the form ax + by + c = 0, where a = π, b = 1 and c = -6.

Example 2: Write each of the following as an equation of the form ax + by + c = 0 and write the values of a, b and c in each case.

  1. x = -3
  2. y = 5
  3. 3x = 2
  4. 5y = 4

Solution:

(i) x = -3 ⇒ x + 3 = 0 ⇒ 1x + 0 x y + 3 = 0. This is of the form ax + by + c = 0, where a = 1, b = 0 and c = 3. (ii) y = 5 ⇒ y - 5 = 0 ⇒ 0x + 1 x y - 5 = 0. This is of the form ax + by + c = 0, where a = 0, b = 1 and c = -5. (iii) 3x = 2 ⇒ 3x - 2 = 0 ⇒ 3x + 0 x y - 2 = 0. This is of the form ax + by + c = 0, where a = 3, b = 0 and c = -2. (iv) 5y = 4 ⇒ 5y - 4 = 0 ⇒ 0x + 5y - 4 = 0. This is of the form ax + by + c = 0, where a = 0, b = 5 and c = -4.

Example 3: Show that x = 3 and y = 2 is a solution of the linear equation 2x + 3y = 12.

Solution:

The given linear equation is 2x + 3y - 12 = 0. ... (i)
Substituting x = 3 and y = 2 in (i), we get,
LHS = (2 x 3 + 3 x 2 - 12) = (6 + 6 - 12) = 0 = RHS.
Therefore, x = 3 and y = 2 is a solution of the equation 2x + 3y = 12.
We may say that (3, 2) is a solution of the equation 2x + 3y = 12.

Example 4: Check which of the following are solutions of the equation x - 2y = 4 and which are not.

  1. (2, 0)
  2. (0, -2)
  3. (4√2, √2)
  4. (4, 0)

Solution:

The given equation may be written as x - 2y - 4 = 0. ...(A)
(i) Putting x = 2 and y = 0 in (A), we get
LHS = (2 - 2 x 0 - 4) = (2 - 0 - 4) = -2 ≠ 0 = RHS.
Therefore, (2, 0) is not a solution of x - 2y = 4.

(ii) Putting x = 0 and y = -2 in (A), we get
LHS = (0 - 2 x (-2) - 4) = (0 + 4 - 4) = 0 = RHS.
Therefore, (0, -2) is a solution of x - 2y = 4.

(iii) Putting x = 4√2 and y = √2 in (A), we get
LHS = (4√2 - 2 x √2 - 4) = (4√2 - 2√2 - 4)
= (2√2 - 4) ≠ 0 = RHS.
Therefore, (4√2, √2) is not a solution of x - 2y = 4.

(iv) Putting x = 4 and y = 0 in (A), we get
LHS = (4 - 2 x 0 - 4) = (4 - 0 - 4) = 0 = RHS.
Therefore, (4, 0) is a solution of x - 2y = 4.

2. Graph of Linear Equation in Two Variable

Method of drawing a graph of ax + by + c = 0, a ≠ =, b ≠ 0

Step 1. Express y in terms of x.
Step 2. Choose some convenient values of x and find the corresponding
values of y satisfying the given equation.
Step 3. Write down these values of x and y in the form of a table.
Step 4. Plot the ordered pairs (x, y) from the table on a graph paper.
Step 5. Join these points by a straight line and extend it in both the directions.
This line is the required graph of the equation ax + by + c = 0.

Examples Regarding The Topic

Example 1: Draw the graph of the equation 2x -y + 3 =0. Using the graph, find the value of y when (a) x = 2, (b) x = - 3.

  1. (2, 0)
  2. (0, -2)
  3. (4√2, √2)
  4. (4, 0)

Solution:

2x - y + 3 = 0 => y = 2x + 3. ... (i)
Putting x=0 in (i), we get y =(2×0+3)=3.
Putting x=1 in (i), we get y =(2×1+3)=5.
Putting x = -1 in (i), we get y =[2×(-1)+3]= 1.
Putting x = -2 in (i), we get y =[2×(-2)+3] = 1

Thus, we have the following table:

x 0 1 -1 -2
y 3 5 1 -1


On a graph paper, draw lines X'OX and YOY' representing the x-axis and the y-axis respectively.
On this graph paper, plot the points A(0, 3), B(1, 5), C(-1, 1) and D(-2,-1). Join AB, AC and CD to get a straight line BACD. Produce it in both ways to get the required graph.

(a) On the x-axis, we take a point P for which x = 2, i.e., OP=2.
From P, draw PE ⊥ X'OX, meeting the graph line BACD produced at E. Then, E(2, 7) shows that,
(x = 2 => y = 7), i.e., when x = 2, then y = 7.

(b) On the x-axis, we take a point Q for which x = -3, i.e., OQ=3 units.
From Q, draw QF ⊥ X'OX, meeting the graph line BACD produced at F. Then, F(-3, -3) shows that,
(x = -3 => y = -3), i.e., when x = -3, then y = -3.
