1. Re-grouping

Process of Re-grouping

In order to regroup in factorization, firstly find the common factor in an equation. But, sometimes in a given expression it is not possible to take out a common factor directly. For example,

 

(i) ab + ax + bc + cx

 

Here, in this expression, we see that, if we try to factorise the expression directly, we will not be able to do so. Therefore, we bring ab and bc together and ax and cx together.

 

Therefore, (ab + bc) + (ax + cx)

 

Now, we take b common from the first group and x common from the second group.

 

= b(a + c) + x(a + c)

= (a + c)(b + x)

 

Hence, we see that we grouped the equation according to the common variables. Thus, this process is called re-grouping.

Examples Regarding The Topic

Example 1: 6ab - b² + 12ac - 2bc

Solution:

= (6ab + 12ac) - b² - 2bc
= (6ab + 12ac) - (b² + 2bc)
= 6a(b + 2c) - b(b + 2c)
= (b + 2c)(6a - b)

 

Example 2: 1 + x + y - z + xy - yz - zx - xyz

Solution:

= (1 + x) + (y + xy) - z - zx - yz - xyz
= (1 + x) + y(1 + x) - z(1 + z) - yz(1 + x)
= (1 + x) [(1 + y) - z(1 + y)]
= (1 + x)(1 + y)(1-z)

Example 3: (2x-3)² - 8x + 12

Solution:

= (2x-3)² - 4(2x-3)
= (2x-3) [(2x-3)-4]
= (2x-3)(2x-7)

2. Factorization using the formula of a² - b²

Understanding the use of the formula a² - b²

Before we learn in-depth information about this topic, we will firstly need to understand the utilisation of the formula a²-b² in the process of factorisation. The formula a²-b² is also known as the difference of squares formula. This formula is use to find the difference between the two squares without actually calculating them.

Verification of a² - b²

The proof that the value a² - b² is (a + b)(a - b). Let us consider the above figure. Take the two squares of sides a units and b units respectively. They can be arranged such that two rectangles are also formed as shown in the above figure.

One rectangle has a length of 'a' unit and a breadth of (a - b) units on the other side the second rectangle has a length of (a - b) and a breadth of 'b' units. Now add the areas of the two rectangles to obtain the resultant values. The respective areas of the two rectangles are (a - b) × a = a(a - b) , and (a - b) × b = b(a - b). The sum of the areas of rectangles is the actual obtained resultant expression i.e., a(a - b) + b(a - b) = (a - b)(a + b). Again re-arranging the individual rectangles and squares, we get: (a + b)(a − b) = a2 − b2.



Let us understand this with a numerical example: x² - y²/100 (Suppose)
Now here, in this equation, we try to put both of the terms into a format of squares.
That is, x² - y²/10 [100 is the square of 10]
Let, x be a,
y/10 be b.
So, following the formula: a² - b² = (a + b)(a - b) we get,
x² - y²/10
= (x + y/10)(x-y/10) [where x = a, y/10 = b]

3. Middle Term factorization

Process of Middle Term factorisation

To factorise a quadratic equation, by splitting the middle term, you can follow these steps:

  • Consider the quadratic equation in the form of ax² + bx + c = 0.
  • Find two numbers that multiply to a and add to b.
  • Split the middle-term using two numbers.
  • Take out the common factors.
  • Simplify.

    4. Expansion and factorization using the formula of (a + b + c)²

    Proving the formula (a + b + c)²

    Proof of (a + b + c)² Formula

    Formula for (a + b + c)² can be proved or expanded in the following two ways:

    Let's discuss these methods in detail as follows:

  • Collecting Like Terms

    • Below we will expand (a + b + c)² by collecting the like terms.
    (a + b + c)² = (a + b + c) (a + b + c)

    ⇒ (a + b + c)² = a (a + b + c) + b (a + b + c) + c (a + b + c)

    ⇒ (a + b + c)² = a² + ab + ac + ba + b² + bc + ca + cb + c²

    From the above expression we will collect all the like terms to get the formula for the (a + b + c)²

    (a + b + c)² = a² + b² + c² + 2ab + 2bc + 2ac

    ⇒ (a + b + c)² = a² + b² + c² + 2(ab + bc + ca)

  • Using Algebraic Identities

    • Below we will expand (a + b + c)² using algebraic identity of (x + y)².

    Let p = b + c

    ⇒ (a + b + c)² = (a + p)²

    By using the algebraic identity (x + y)² = x² + y² + 2xy we get

    (a + b + c)² = a² + p² + 2ap

    Now putting the value of p in the above expression we get

    (a + b + c)² = a² + (b + c)² + 2a (b + c)

    Now again using the identity (x + y)² = x² + y² + 2xy expand (b + c)²

    (a + b + c)² = a² + b² + c² + 2bc + 2a (b + c)

    ⇒ (a + b + c)² = a² + b² + c² + 2bc + 2ab + 2ac

    ⇒ (a + b + c)² = a² + b² + c² + 2(ab + bc + ac)

    8. Factorization using Vanishing Method

    What is the Vanishing method and how is it used?

    Vanishing method or trial method of factorisation of any given polynomial is a method of determining the factor (or factors) of the polynomial by finding its zero (or zeroes). This zero (or zeroes) is determined by experiment or trial, i.e., the values of the variable (or variables) for which the value of the given polynomial is zero, are determined by experiments or trials. For example, let f(x) = x² + 2x - 3 be a given polynomial, we have to find its zero (or zeroes), i.e., we want to determine the values of the variable x for which the value of f(x) is zero.

    Now putting x = 0 in f(x) we get, f(0) = 0² + 2 * 0 - 3 = -3 not equal to 0.

    Putting x = 1 in f(x) we get, f(1) = 1² + 2 * 1 - 3 = 1 + 2 - 3 = 0

    Therefore, x = 1 is a zero of the polynomial f(x) = x² + 2x - 3.

    Again, putting x = 2 in f(x) we get, f(2) = 2² + 2 * 2 - 3 = 4 + 4 - 3 = 5 not equal to 0.

    Putting x = 3 in f(x) we get, f(3) = 3² + 2 * 3 - 3 = 9 + 6 - 3 = 12 not equal to 0.

    Putting x = -3 in f(x) we get, f(-3) = (-3)² + 2 * (-3) - 3 = 9 - 6 - 3 = 0.

    Therefore x = -3 is another zero of the polynomial f(x) = x² + 2x - 3.

    Thus, the zeroes of the given polynomial are determined by putting the different values of the variable in the given polynomial on the trial basis.

    Now, the question is that is there any easy rule or method of finding the exact values of the variable for which the value of the polynomial is zero ? In reply, it can be said that:

    (i) if the co-efficient of the term, containing the highest power of the variable of the polynomial be 1, then the zero of this polynomial will be any one of the factors of the constant term of the polynomial

    Such as, the coefficient of x², the term containing the highest power of x, in the polynomial f(x) = x² + 2x - 3 is 1, therefore the zeroes of f(x) will be any one of (±1) and (±3), which are the factors of the constant term 3 of f(x). We have already seen in the above that (+1) and (-3) are two zeroes of f(x) = x² + 2x - 3.

    (ii) If the co-efficient of the term containing the highest power of the variable in the given polynomial be a and the constant term be "c", then the zeroes of the polynomial are either the factors of "a" or the factor of "c" or the numbers obtained by the division of factors of "a" and factors of "c" and factors of "c" and factors of "a"

    For example, let f(x) = 2x² + x - 3 be a polynomial. We have to factorise it. Then, the co-efficient of the term containing the highest power of the variable x in the polynomial is 2, the factors of which are ± 1 and ± 2. Also, the factors of the constant term 3 are ± 1 and ± 3. So, the zeroes of f(x) must lie amongst the number ± 1, ± 2, ± 3, ± 1/3, ± 2/3. Generally, taking these values of x if, trials are made successively, we shall get the zeroes of f(x).

    Now, x^4 + 3x³ + 3x² + 2x + 1 = x³(x + 1) + 2x²(x + 1) + x(x + 1) + 1(x + 1)
    = (x + 1)(x³ + 2x² + x + 1)
    Ex. 2. Factorise (using factor theorem) : x³ - x² - (2 - √{2})x + √{2}

    Solution:

    Let, f (x) = x³ - x² - (2 - √{2})x + √{2}
    f(-1) = (-1)³ - (-1)² - (2 - √{2}) (-1) + √{2} = -1 - 1 + 2 - √{2} + √{2} = 0.

    Therefore, (x + 1) is a factor of f (x)
    Now, x³ - x² - (2 - √{2})x + √{2} = x²(x + 1) - 2x(x + 1) + √{2}(x + 1) = (x + 1)(x² - 2x + √{2})
